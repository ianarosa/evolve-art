🎨 What am I watching?

The picture on the right is built from a few dozen colored glass shapes stacked on top of each other. It starts as random junk.

The computer keeps making tiny random tweaks — nudge a corner, change a color — and only keeps a change if it makes the picture look MORE like the target on the left. Bad tweaks are thrown away.

Over thousands of these tiny improvements, random shapes slowly evolve into a copy of the image. That's evolution as an artist — and it's why the Match % only ever goes up.

How it works

The picture's "genome" is a list of colored, semi-transparent polygons. Each attempt copies the current best, makes one small random change, and redraws it. If the new version matches the target better, it becomes the new best; otherwise it's discarded.

Because only improvements are kept, the Match % can never go down. Fitness is measured on a small downscaled copy for speed, then the winner is painted large on the right.
